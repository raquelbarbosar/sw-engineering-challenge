## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
[Docker](https://www.docker.com) container ecosystem for database.

## Project setup

```bash
$ npm install
```

## Start database

```bash
$ npm run db:start
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
